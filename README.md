#  Separation of Recyclable & Non-Recyclable Papers  

##  Overview  
An intelligent system designed to automatically classify and separate recyclable and non-recyclable paper materials using sensors and a microcontroller, improving waste management efficiency and supporting environmental sustainability.  

---  

##  Features  
-  Automatic paper classification  
-  Sensor-based detection system  
-  Motorized sorting mechanism  
-  LED indicators for status  
-  Promotes eco-friendly waste segregation  

---  

##  Components Used  
- Arduino / Microcontroller  
- Moisture Sensor / dry or wet waste separation  
- IR Sensor / Recyclable Paper
- Colour Sensor / Non-Recyclable Paper  
- Servo Motor / DC Motor  
- Conveyor mechanism  
- Stepper Motor 
- Power Supply  

---  

##  Working  
The system detects the type of paper using sensors (such as moisture or texture detection). Based on the detected properties, the microcontroller processes the data and activates the motor to direct the paper into either the recyclable or non-recyclable bin.
